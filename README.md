# Personal Finance Manager App

A simple and interactive web application built with React and Vite to help track daily financial activities. This dashboard provides insightful analytics, transaction logging, and a cohesive user interface for managing personal budgets.

## Features Added

- **Secure Data Tracking:** Easily log expenses and incomes securely.
- **Dynamic Dashboard:** Real-time summary views of total balance, total income, and expenditures.
- **Visual Analytics:** Breakdown of expenses by category with visual representations.
- **User-Friendly Interface:** Built using modern styling with Tailwind CSS.
- **State Management:** Utilizes React's native Context API to maintain global user and financial data state.

## Technologies Used

- React 19
- Vite (for fast tooling and building)
- React Router DOM (client-side routing)
- Tailwind CSS (atomic CSS architecture)
- Lucide React (vector icons)
